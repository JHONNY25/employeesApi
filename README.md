# Empleados

## Instrucciones

Crear db "employees"

## Correr migraciones

```bash
php artisan migrate
```

Ejecute los seeders

```bash
php artisan db:seed
```

## Instalar dependencias npm

```bash
npm i --save
```

## Iniciar servidor artisan

```bash
php artisan serve
```

## luego

```bash
npm run watch
```
